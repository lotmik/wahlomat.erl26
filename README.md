# Wahl-O-Mat Erlangen 2026

Den Wahl-O-Mat findest du hier:
https://lotmik.github.io/wahlomat.erl26/

So funktioniert er:
1. Link öffnen und auf **„Jetzt starten“** klicken.
2. Die Thesen mit **Ja / Neutral / Nein** beantworten (optional doppelt gewichten).
3. Am Ende die Übereinstimmung mit den Parteien und die Zitate vergleichen.
